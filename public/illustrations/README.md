# Brand Illustration Library

Standalone SVG assets for reusable NorthSpark brand illustrations.

- `trust.svg`
- `message.svg`
- `browser.svg`
- `shopping.svg`
- `strategy.svg`
- `cursor.svg`
- `shield.svg`
- `document.svg`
- `phone.svg`

Each illustration uses a transparent background, a `160 x 160` viewBox, thin rounded outline strokes, a muted sage rectangle, a soft peach circle, and a subtle dotted pattern.
