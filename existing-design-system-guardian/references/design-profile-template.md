```markdown
- 
```

## Confidence Guidance

Use `High` when multiple repo sources agree and the affected surface has clear precedents.

Use `Medium` when there is enough local evidence to proceed, but broader product conventions are incomplete.

Use `Low` when evidence is sparse, conflicting, or unavailable. Ask for clarification before making visually directional choices.
